
Welcome to the angularjs-openweather with Bootstrap !

This a single page application. This is simple, type the City name to request weather 

How to use and run this application ?

Requirements :
Node.js with npm.
Favorite software to unzip/uncompress file.

First action : 
Install the Node.js with npm module, from offical web site https://nodejs.org (version v6.3.1 or Current)

Secound action:

Clone from github or Download the project and unzip.

Third action :

Open your favorite console and enter in unziped dir and run command npm install (to install the dependencies) after run command npm start to start webserver node and deploy application in http://localhost:3000 (Yes port 3000 because prevent conflit)

Bem vindo à Aplicação de Verificação do Clima com AngularJs e Bootstrap utilizando a API do OpenWeatherMap

Esta é uma Aplicação de página única (single page) , simples com o intuito de verificar o clima na cidade pesquisada.

Como usar esta aplicação ?

Requisitos :
Node.js
Programa para descomprimir arquivos.(Ex.: Winrar, Winzip)

Passo 1:
Instalar o Node.js com o pacote npm (https://nodejs.org) versão v6.3.1 ou a Corrente/Atual

Passo 2: 
Clonar este projeto via git/github ou realizar o Download e descompactar o arquivo.

Passo 3:
Iniciar seu programa de prompt de comando favorito.
Entrar na pasta que descompactou e executar o comando: npm install (para baixar possíveis dependencias) 
e após executar o comando npm start (para iniciar o servidor)
com o deploy da aplicação que ficará disponível em http://localhost:3000
